# ObysAgency Website
A modern, interactive web agency portfolio built with vanilla HTML, CSS, and JavaScript.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Locomotive Scroll
- GSAP
- Three.js
- Shery.js

## Project Structure
```
├── index.html          # Main HTML file
├── style.css           # Stylesheet (required)
├── script.js           # JavaScript functionality (required)
├── Assets/
│   ├── favicon-32x32.png
│   ├── img1-1.jpg through img4-2.jpg (project images)
└── README.md
```

## Features
- Smooth scrolling animations
- Custom cursor effects
- Video showcase section
- Project gallery with hover effects
- Animated loading screen
- Responsive design
